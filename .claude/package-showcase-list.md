# Package Showcase List

Reference list of packages to build interactive showcase pages for.

---

## Animation & Motion

- **Motion One** — Web Animations API based, tiny and performant
- **GSAP** — Industry standard, timelines, scroll-triggered animations
- **AutoAnimate** — Zero-config automatic child element transitions
- **Lenis** — Smooth scrolling library, changes how a page feels
- **Splitting.js** — Break text/elements into individually animatable pieces
- **Flipping.js** — FLIP animation technique helper
- **Popmotion** — Functional, reactive animation library (powers Framer Motion)
- **CountUp.js** — Animated number counting
- **Typed.js** — Typewriter / typing animation effect
- **Rough Notation** — Hand-drawn style annotation animations

## Visual & Creative

- **Three.js** — 3D rendering engine
- **Zdog** — Pseudo-3D with flat, illustrative shapes
- **Paper.js** — Vector graphics scripting framework
- **p5.js** — Creative coding (Processing for the web)
- **Pts.js** — Points, lines, shapes for creative coding
- **Fabric.js** — Interactive canvas with object model
- **PixiJS** — Fast 2D WebGL renderer
- **Two.js** — 2D drawing API, renderer agnostic (SVG, Canvas, WebGL)

## Interaction & Gesture

- **Hammer.js** — Touch gesture recognition (pinch, rotate, swipe, press)
- **interact.js** — Drag, resize, snap, multi-touch element manipulation
- **Sortable.js** — Drag-and-drop reorderable lists
- **Panzoom** — Pan and zoom elements with touch/mouse support
- **use-gesture** — Gesture handling (works standalone, not React-only)

## Data & Visualization

- **D3.js** — The big data visualization library
- **Observable Plot** — Simpler, opinionated charting from the D3 team
- **Chart.js** — Simple, clean charts with good defaults
- **Vega-Lite** — Declarative visualization grammar (JSON spec)
- **Frappe Charts** — Simple, responsive SVG charts
- **uPlot** — Tiny, fast time series charts

## State & Data Management

- **Zustand** — Tiny state management, no boilerplate
- **Immer** — Write mutating code that produces immutable state
- **Dexie.js** — IndexedDB wrapper that feels like a real database
- **localForage** — Unified API across localStorage, IndexedDB, WebSQL
- **RxJS** — Reactive programming with observables
- **Nanostores** — Tiny state manager, framework agnostic

## Utilities

- **Zod** — Runtime type/schema validation
- **Fuse.js** — Fuzzy search, typo-tolerant
- **nanoid** — Tiny unique ID generator
- **hotkeys-js** — Keyboard shortcut binding
- **date-fns** — Modular date utilities
- **Luxon** — Modern date/time with timezone and locale support
- **Lodash** — General utility belt (debounce, throttle, deep clone, etc.)
- **uuid** — RFC-compliant UUID generation
- **change-case** — Convert between camelCase, snake_case, etc.
- **ms** — Convert time strings to milliseconds and back ("2 days" → ms)

## Parsing & Transformation

- **Cheerio** — HTML parsing and manipulation
- **Marked** — Markdown to HTML
- **markdown-it** — Markdown parser with plugin ecosystem
- **Papa Parse** — CSV parsing and generation, handles edge cases
- **Turndown** — HTML to Markdown conversion
- **DOMPurify** — Sanitize HTML to prevent XSS
- **js-yaml** — YAML parsing and serialization
- **Shiki** — Syntax highlighting using VS Code's engine

## Networking & HTTP

- **ky** — Tiny HTTP client built on fetch, cleaner API
- **Socket.IO** — Real-time bidirectional communication (client side)
- **sse.js** — Server-Sent Events client
- **Wretch** — Tiny fetch wrapper with fluent API

## Audio

- **Tone.js** — Web Audio framework for music and sound synthesis
- **Howler.js** — Audio library for games and apps
- **Pizzicato.js** — Simplified Web Audio API with effects

## Layout & UI

- **Masonry** — Cascading grid layout (Pinterest style)
- **Muuri** — Responsive, sortable, filterable grid layouts
- **Tippy.js** — Tooltip and popover positioning
- **Floating UI** — Positioning engine for tooltips, popovers, dropdowns
- **Swiper** — Modern touch slider / carousel

## Text & Typography

- **Rough.js** — Hand-drawn style graphics
- **Textfit** — Auto-fit text to container
- **Fitty** — Snugly resize text to fit its parent
- **Mark.js** — Text highlighting / search term marking
