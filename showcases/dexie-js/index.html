Dexie.js

dexie.org/docs →

A minimalistic IndexedDB wrapper that makes browser storage effortless. Promises-based API, versioned schemas, and excellent error handling.

State/Data
Why it's interesting: IndexedDB is powerful but its raw API is callback-heavy and awkward. Dexie wraps it with a clean, promise-based interface that feels like a real database. Schema versioning, compound indexes, and transactions come built in. It's perfect when localStorage isn't enough and you need structured, queryable client-side storage.

Persistent Notes

A mini note-taking app backed by IndexedDB. Add notes, delete them, refresh the page — they persist.

    View Code 
    import Dexie from "dexie";

const db = new Dexie("ShowcaseNotesDB");
db.version(1).stores({
  notes: "++id, text, createdAt"
});

// Add a note
await db.notes.add({
  text: "Hello world",
  createdAt: new Date()
});

// Get all notes (newest first)
const all = await db.notes
  .orderBy("createdAt")
  .reverse()
  .toArray();

// Delete a note

    View Code 
    import Dexie from "dexie";

const db = new Dexie("ShowcaseCounterDB");
db.version(1).stores({
  kv: "key"  // simple key-value store
});

// Read the counter
const entry = await db.kv.get("counter");
let count = entry ? entry.value : 0;

// Update the counter (upsert)
await db.kv.put({ key: "counter", value: count });
