Fuse.js

fusejs.io →

Lightweight fuzzy-search library with zero dependencies. Perfect for client-side search without a server.

Utilities
Why it's interesting: Fuse.js brings powerful fuzzy searching to the client with no backend. It handles typos, partial matches, and weighted multi-key search. It's the go-to when you need "search that just works" for small-to-medium datasets without setting up Elasticsearch or Algolia.

Live Fuzzy Search

Type to fuzzy-search through a list of popular JS packages. Try typos — Fuse handles them.

    View Code 
    import Fuse from "fuse.js";

const fuse = new Fuse(packages, {
  keys: ["name", "description"],
  includeScore: true,
  includeMatches: true,
  threshold: 0.4
});

input.addEventListener("input", (e) => {
  const results = fuse.search(e.target.value);
  // results = [{ item, score, matches }, ...]
});
